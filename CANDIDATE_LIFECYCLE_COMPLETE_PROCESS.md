## **DATABASE STATUS TRACKING THROUGHOUT LIFECYCLE**

```sql
-- Candidate Status Progression
CREATE TABLE candidate_status_history (
  id SERIAL PRIMARY KEY,
  candidate_id VARCHAR(50),
  status_from VARCHAR(50),
  status_to VARCHAR(50),
  timestamp TIMESTAMP,
  trigger_event VARCHAR(100),
  notes TEXT
);

-- Status Values Throughout Lifecycle:
REGISTERED          → Initial registration complete
DOCUMENTS_PENDING   → Awaiting document uploads
AI_ASSESSMENT       → Taking assessments
ASSESSED           → All assessments complete
AVAILABLE          → Ready for client review
MATCHED            → Matched to job opportunity
INTERVIEWING       → In interview process
OFFERED            → Received job offer
NEGOTIATING        → In salary negotiation
HIRED              → Accepted offer, contract signed
PRE_BOARDING       → Preparing for start date
ONBOARDING         → First 90 days
EMPLOYED           → Fully integrated employee
PROMOTED           → Career advancement
RESIGNED           → Notice period
EXITED             → Exit interview complete
ALUMNI             → Former employee network
```
